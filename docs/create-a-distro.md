# Creating a Little Island distro

In order to do advanced modding, you need to be able to create your own distro (version of the game).

These distros allow you to share your work with others, as well as simply add features that didn't already exist to the game.

> Note: In order to create distros of the game, you need to understand **Java**.

> Consider learning the language before modding the game in this way.

> Alternatively, you can [mod Little Island without coding](mods.md).

## Setup

To start, try compiling the standard game into a `.jar` file.

Once you have done that, you can now [start adding features](#adding-features)!

## Adding features

To add or modify features to the game, simply edit the `.java` files!

You may need to modify some existing files to get everything working. Check out the [distro maker's guide](distro-maker-guide.md) for help on adding features to the game!

Make sure to make every feature of your distro mod-friendly.  
This means making sure all features are editable using `.li` files.
